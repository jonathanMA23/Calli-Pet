# Sistema de diseño de Calli Pet

## Personalidad

- Cercana
- Confiable
- Ordenada
- Cálida
- Profesional
- Mexicana de forma sutil

## Paleta

| Token | Valor | Uso |
|---|---|---|
| Negro principal | `#171717` | Texto y acciones |
| Marfil | `#F7F3EA` | Fondo |
| Blanco | `#FFFFFF` | Superficies |
| Terracota | `#A64B2A` | Marca y selección |
| Olivo | `#5E6B4F` | Estados favorables |
| Gris | `#6B6B6B` | Texto secundario |
| Rojo | `#B42318` | Alertas |

## Escala

- Auxiliar: 12–14 px
- Texto: 16 px
- Subtítulo: 20–24 px
- Título: 32–40 px
- Hero: 48–64 px

## Espaciado

Unidad base de 4 px. Usar 8, 12, 16, 24, 32 y 48 px.
